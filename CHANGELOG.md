# Changelog

All notable changes to causal-edge are documented here.

## [0.1.0] - 2026-04-02

### Added
- **Framework core**: StrategyEngine ABC, config loader, CLI (init/run/dashboard/validate/status)
- **Abel Proof validation**: 15-test gate with anti-gaming metric triangle (Lo-adjusted Sharpe, IC, Omega)
- **Dashboard**: Dark-theme static HTML with Plotly equity curves and position charts
- **Demo strategies**: SMA crossover (simple) and Momentum ML (walk-forward GBDT)
- **Agent-native architecture**: AGENTS.md decision trees at every subsystem, 15 structural tests
- **Project scaffolding**: `causal-edge init` creates harness-contagious project skeleton
